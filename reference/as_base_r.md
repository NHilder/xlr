# Convert xlr types to their base R type

`as_base_r` converts xlr objects,
[xlr_table](https://nhilder.github.io/xlr/reference/xlr_table.md),
[xlr_numeric](https://nhilder.github.io/xlr/reference/xlr_numeric.md),
[xlr_integer](https://nhilder.github.io/xlr/reference/xlr_integer.md),
[xlr_percent](https://nhilder.github.io/xlr/reference/xlr_percent.md),
and [xlr_format](https://nhilder.github.io/xlr/reference/xlr_format.md)
to their base R type.

## Usage

``` r
as_base_r(x)
```

## Arguments

- x:

  a xlr object

## Value

The base type of the base R object.

## Details

as_base_r is a generic. It is a wrapper around
[vec_data](https://vctrs.r-lib.org/reference/vec_data.html) but will
convert every object to its base type.
